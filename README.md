# Pulse

Pulse is an Android music player focused on offline listening and a clean user experience.

## Features

- Offline music playback
- Mini player
- Media3 player engine
- Shuffle mode
- Repeat modes
- Permission handling

## Built with

- Java
- Android SDK
- Media3 ExoPlayer
- RecyclerView
- Material Components

## Installation

1. Download the latest APK from Releases.
2. Open the APK file on your Android device.
3. Android may show a security warning because Pulse is distributed outside Google Play.
4. Allow installation and open the app.
