# Pulse

Pulse is an Android music player focused on offline listening and a clean user experience.

## Features

- Offline music playback
- Mini player
- Media3 player engine
- Shuffle mode
- Repeat modes
- Permission handling

## Built With

- Java
- Android SDK
- AndroidX Media3 (ExoPlayer)
